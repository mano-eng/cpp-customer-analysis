# Customer Analysis System (C++)

A C++ console application that processes customer data over a 30-day period and performs basic statistical analysis.

---

##  Features

- Calculates daily income when number of kids exceeds number of men.
- Tracks the maximum total time (service duration + waiting time).
- Uses structured looping and conditional logic.

---

##  Technical Concepts

- Nested loops
- Conditional statements
- Counters and accumulators
- Basic algorithmic thinking
- Input validation structure

---

##  Input Format

For each day:
- Number of customers

For each customer:
- Type: K (Kid) or M (Man)
- Service duration
- Waiting time
- Price

---

##  Output

- Daily income (only if kids > men)
- Maximum total time across all customers
